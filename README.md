# DLMC-QUANT
This is a model quantization tool, including some basic quantization algorithms.
I also release our works [RootQ](https://link.springer.com/article/10.1007/s10489-022-03691-1) and [RepAPQ]() here.
## RootQ
Using root-base estimator to enhance QAT accuracy.
## RepAPQ
Resolve accuracy drop for RepVGG and MobileOne models within several minutes.
